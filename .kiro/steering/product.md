# Product Overview

Briefly Cloud is a cloud-native AI productivity assistant that transforms documents into intelligent conversations. It's a desktop application that connects to cloud storage (Google Drive, OneDrive) and uses AI to help users interact with their documents through natural language.

## Core Value Proposition
- Upload documents from cloud storage and ask questions about them
- AI-powered chat using GPT-4o with semantic search across document knowledge base
- Progressive Web App with offline support and mobile-ready interface
- Subscription-based SaaS with multiple tiers (Free, Pro, Team, Enterprise)

## Key Features
- **AI Chat**: Intelligent conversations about documents using GPT-4o
- **Multi-format Support**: PDF, DOCX, TXT, MD, CSV, XLSX, PPTX
- **Vector Search**: Semantic search across entire knowledge base using ChromaDB
- **Cloud Integration**: Google Drive and OneDrive OAuth integration
- **Secure Authentication**: OAuth 2.0 with JWT tokens
- **Subscription Billing**: Stripe-powered billing system
- **Mobile PWA**: Progressive Web App with service workers

## Target Users
Knowledge workers, researchers, and teams who need to quickly extract insights from large document collections through conversational AI interface.

## Business Model
Freemium SaaS with tiered subscriptions:
- Free: 10 docs, basic features
- Pro ($30/month): 1,000 docs, GPT-4o
- Pro BYOK ($15/month): 10,000 docs, your own API key
- Teams/Enterprise: Coming Soon - collaboration, custom AI, priority support
